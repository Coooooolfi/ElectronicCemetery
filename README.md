# Electronic Cemetery – информационная система учёта захоронений

## Назначение
Приложение предназначено для автоматизации учёта кладбищ, секторов, участков, захоронений и усопших. Поддерживаются роли: Посетитель (просмотр и поиск), Администратор (управление секторами и участками на закреплённом кладбище), Супер-администратор (полный контроль).

## Установка и запуск
1. Установите PostgreSQL 13+ и создайте базу данных `project_cemetery`.
2. Выполните SQL-скрипт из файла `schema.sql` (создание таблиц и заполнение начальными данными).
3. Настройте параметры подключения в классе `DbConnection.java` (URL, логин, пароль).
4. Соберите проект с помощью Maven: `mvn clean package`
5. Запустите приложение: `java -jar target/electroniccemetery-1.0-SNAPSHOT.jar`

## Использование
- **Посетитель:** нажмите «Посетитель» на стартовом экране → навигация по кладбищам, поиск по ФИО.
- **Администратор:** логин `admin1`, пароль `12345` (управление секторами и участками).
- **Супер-администратор:** логин `superADMIN`, пароль `superpuper` (полный доступ).
